# Ratio & Proportion Quick Sheet for Aptitude Tests

## 1. What is a Ratio?

A ratio compares two quantities.

If a : b then

a / b

Example:

3 : 5 means
3/5

---

# Core Concepts

## 1. Simplifying Ratios

Divide both numbers by their HCF.

Example:

18 : 24

HCF = 6

18/6 : 24/6

= 3 : 4

---

## 2. Dividing a Quantity in a Ratio

If a quantity is divided in the ratio a : b

Total parts = a + b

First part = Total × a/(a+b)

Second part = Total × b/(a+b)

Example:

900 divided in ratio 2 : 3

Total parts = 5

First = 900 × 2/5 = 360

Second = 900 × 3/5 = 540

---

## 3. Equivalent Ratios

Multiply both terms by same number.

Example:

3 : 4

= 6 : 8

= 9 : 12

---

## 4. Proportion

If

a : b = c : d

Then

a × d = b × c

Example:

3 : 5 = 9 : x

3x = 45

x = 15

---

# Important Tricks

## Trick 1: Scaling Ratio

If ratio is a : b

Actual values can be

a×k : b×k

Example

7 : 9

= 7k : 9k

---

## Trick 2: Adding Quantities

If A : B = 3 : 5

Total parts = 8

If total = 160

A = 160 × 3/8

B = 160 × 5/8

---
